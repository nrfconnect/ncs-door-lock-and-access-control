Wireless technologies
#####################

The |APP_NAME| combines multiple wireless technologies to deliver a complete door lock and access control reference.
This page describes each technology used across the add-on and how it integrates with the rest of the application.

The following wireless technologies are supported:

* :ref:`wireless_technologies_aliro` — Standardized access protocol for phone-as-a-key user experience.
* :ref:`Near Field Communication (NFC) <wireless_technologies_nfc>` — Short range communication for tap-to-unlock access.
* :ref:`wireless_technologies_matter` — Universal smart home protocol for command-and-control and credential provisioning.
* :ref:`wireless_technologies_ble` — General-purpose Bluetooth LE services (SMP, NUS) that operate independently of Aliro and Matter.
* :ref:`Ultra wideband (UWB) <wireless_technologies_uwb>` — Proximity-based access with hands-free unlock.
