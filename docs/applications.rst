Applications
############

The |REPO_NAME| delivers reference applications.
Each one is a complete firmware image you can build, flash, and test on a supported development kit.
Pages in this section cover configuration, build and run instructions, testing, firmware update, and troubleshooting for the application you choose.

Wireless transports, protocol integration, and shared modules are described in :ref:`wireless_technologies` and :ref:`components`.
To compare the applications and pick a starting point, see :ref:`solution_overview`.
